# aix-admin

#### 介绍
aix智能后台管理系统

采用前后端分离的模式，前端开源两个框架：aix-react(基于 React、Ant Design)、aix-vue (基于 Vue、Element-UI)。
后端分为框架实现aix-framework, boot版本aix-admin, cloud版本aix-admin-cloud  
借鉴ddd领域驱动，增加领域层，适用于复杂项目开发  



工程结构
aix-framework  
├── aix-boot -- boot相关  
├── aix-cloud -- cloud相关  
├── aix-common -- 工具相关  
├── aix-modules -- 组件模块  
├── aix-db -- 持久层模块  
├── aix-web -- web模块  
├── aix-security -- 安全模块
